# How to Apply #

01Loging

Requirements

  • HSC: Roll, Board
  • SSC: Roll
02Details

Requirements

  • Question & Answer Script Language
  • Current Address Details
  • Phone No, Email ID ( Must be personal)
  • Parent NID Number (Optional) and Occupation
  • Available QUOTA (Detail in instructions)
  • Choice of Exam Center (Division)

Requirements

  • Width: 360-540 px
  • Height: 540-720 px
